## Usage Notes

1. **Without Panels:** Use when customer has existing wall structure or is building custom framework
2. **With Panels:** Use when DDS wall panels will be installed (most common scenario)
3. **Default Width:** Always assume 30" wide doors unless otherwise specified
4. **Heights Available:** 67", 75", and 79" (79" is most common for standard installations)
5. **Measurement:** All dimensions are for continuous lineup configurations
